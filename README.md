# House Price Prediction using Machine Learning

This project builds a Machine Learning model to predict house prices using the California Housing dataset available in the Scikit-learn library. The model is trained using the XGBoost Regressor algorithm and evaluated using regression metrics.

The project demonstrates a complete machine learning workflow including data analysis, visualization, model training, and prediction.

---

## Project Overview

House price prediction is a common regression problem in machine learning. This project uses the California Housing dataset and applies machine learning techniques to estimate house prices based on different features such as income, house age, population, and location.

---

## Dataset

The dataset used in this project is the **California Housing Dataset** from the Scikit-learn library.

Features used in the dataset include:

- MedInc (Median Income)
- HouseAge
- AveRooms
- AveBedrms
- Population
- AveOccup
- Latitude
- Longitude

Target Variable:

- House Price

---

## Project Workflow

1. Import required libraries
2. Load California Housing dataset
3. Convert dataset to Pandas DataFrame
4. Perform Exploratory Data Analysis (EDA)
5. Generate correlation heatmap
6. Split dataset into training and testing sets
7. Train model using XGBoost Regressor
8. Evaluate model performance
9. Compare actual vs predicted values
